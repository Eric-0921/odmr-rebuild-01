## 二、全局修改类别

1. **标题层级规范化**：顶层统一为 `# N. 标题`，二级 `## N.M ...`，三级 `### ...`，四级 `#### ...`；删除孤立/跳级标题（如原 `## 1、RJ45 网络接口`、`## 4. 完成以上操作之后...`、`## 1/f 噪声` 等）。
2. **编号纠错**：修正 `3.20E1300` → `3.2 OE1300`、`3. 17` → `3.17`、`5.10E1300` → `5.1 OE1300`；7.1 步骤 4 已归位；4.3 下 RJ45/串口已整理为表格并按 4.3.1/4.3.2 编号。
3. **公式与单位**：所有物理量与单位使用 LaTeX `\mathrm{}` 包裹（如 `$10\ \mathrm{M\Omega}$`、`$5\ \mathrm{nV/\sqrt{Hz}}$`），独立公式使用 `$$...$$`；修正原文中 `$\theta$` 显示为 `0` 等 OCR 错误。
4. **表格化**：技术参数、安全注意事项、接口说明、全部 SCPI 命令参考等由零散段落/OCR HTML 表格整理为标准 Markdown 表格。
5. **图片保留与规范化**：保留全部 78 张内容图片（原 PaddleOCR 共 79 张，封面 LOGO 图片未纳入正文），图注统一为 `<div style="text-align: center;">图 N. ...</div>`，全文图号顺序重排为 1–77。
6. **页眉/页脚清理**：删除被误识别的 `OE1300 Lock-In Amplifier`、`DSP Lock-In Amplifier` 等页眉文字。
7. **格式细节**：第 7 章中 `$\underline{图NN}$`、`$\uwave{\text{断开}}$` 等 CJK 文本嵌入 LaTeX 的写法统一改为 HTML `<u>...</u>`；修复嵌套 `<div style="text-align: center;">`。
