## 5. 跨文档约定

1. **指令严格沿用说明书原文**:`FMODD`、`ISRCD`、`OEXPD` 等非标准 SCPI 风格,**不做改写**。
2. **终止符**:使用 `<lf>`(`\n`)或 `<cr>`(`\r`),与说明书一致。
3. **多指令同行**:用 `;` 分隔,如 `ISRCD 1,0;IGNDD 1,0 <lf>`(注意整个命令行 ≤ 256 字符)。
4. **查询语法**:在命令后加 `?` 即可,参数省略。如 `FMODD? 1 <lf>`。
5. **通道参数 i**:`1` = 通道 A,`2` = 通道 B,与说明书 5.1 章节定义一致。
6. **截图灰显项**:在 UI 上因前置条件不满足而禁用,但**远程指令无此限制**,可正常下发。文中已逐一标注。

## 6. 已知指令/手册疑误(写代码时需注意)

| 出处 | 原文问题 | 建议处理 |
| --- | --- | --- |
| 5.2.1 SVTMD 行 | 把指令名写成了 `SVULD` | 以指令表格中的 `SVTMD` 为准 |
| 5.2.5 OEXPD 语法行 | 写为 `OEXPD (?) j,k{, x, k}`,实际第 4 参数应是 `l` | 实际格式:`OEXPD j,k,x,l` |
| 5.2.8 STRGD 行 | 语法写为 `STRGD`,正文第一处写作 `STRFGD` | 以语法行 `STRGD` 为准 |
| 4.2.2 提到 `Channel-A` | 仅前面板可选,远程指令无对应 j 值 | 远程方式不支持,需面板操作 |
| 4.2.3 提到 `TTL Neg` | 远程指令 `RSLPD` 仅支持 j=0/1 | 远程方式不支持 |
