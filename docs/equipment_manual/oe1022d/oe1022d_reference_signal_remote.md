# OE1022D 参考信号 — 配置项 ↔ 远程指令映射(Ch-A + Ch-B)

> **范围**:截图"参考信号配置"标签页(对应说明书第 4.2 节)
> **指令口径**:严格区分 `手册原文` 与 `真机观测`
> **通道参数 i**:`1` = 通道 A,`2` = 通道 B

## 2.1 参考相位调整

| 截图配置项 | 截图当前值 | 远程指令 | 设置语法 | 查询语法 |
| --- | --- | --- | --- | --- |
| 参考相位调整(°) | 0.00 (Ch-A & Ch-B) | `PHASD` | `PHASD i,x <lf>` | `PHASD? i <lf>` |

### 参数说明

- `i` = 通道(1 = A, 2 = B)
- `x` = 相位值(实数,单位 °)
- 范围:-180.00° ~ +180.00°
- 精度:0.01°(四舍五入)

**配置示例**:
```text
PHASD 1,-179.0 <lf>    ; Ch-A 设为 -179.00°
PHASD? 1 <lf>          ; 返回 "-179.00"
PHASD 2,90.5 <lf>      ; Ch-B 设为 90.50°
```

## 2.3 外部参考信号类型

| 截图配置项 | 截图当前值 | 远程指令 | 设置语法 | 查询语法 |
| --- | --- | --- | --- | --- |
| 外部参考信号类型 | Ch-A 灰显 / Ch-B = 过零检测 | `RSLPD` | `RSLPD i,j <lf>` | `RSLPD? i <lf>` |

### 手册原文枚举（V1.5 PDF）

| j | 含义 | 截图对应 |
| ---: | --- | --- |
| 0 | TTL 上升沿触发 | — |
| 1 | 正弦波过零检测(Sine Zero Detect) | Ch-B 当前值 |

> 校正说明（2026-06-11）：原始 PDF 页面 28 的 `4.2.3 <Ref.Slope>` 图面只展示 `TTL` 与 `Sine` 两类，不支持本仓库早期整理稿中写入的 `TTL Neg` 说法。
> 当频率低于 1Hz 时,必须使用 TTL 触发方式(j=0)。

### 真机观测（厂商软件配置后）

在 `2026-06-11` 的真机只读快照中，厂商 LabVIEW 软件将通道 B 配置到 PLL 锁定后，观测到：

| 查询 | 观测值 | 说明 |
| --- | --- | --- |
| `FMODD? 2` | `0` | 外部参考 |
| `RSLPD? 2` | `2` | 该值**不在 V1.5 PDF 的 `0/1` 枚举内** |
| `FREQD? 2` | `4.99999e+02` | OE 已读到约 500 Hz 外部参考 |
| `*PLLD? 2` | `1` | PLL 已锁定 |

对应产物：

- `out/hardware_state_snapshot/manual_after_labview/hardware_state_snapshot.json`

当前结论：

- `RSLPD` 的 `0/1` 说明可作为 `manual-stated` 层保留
- 但后续 helper / profile / runtime 不得再假设“真实设备只可能回 `0/1`”
- 在没有受控 write-back 实验之前，`RSLPD? 2 = 2` 只记为已观测真值，不强行推断语义

## 2.4 内部参考信号频率

| 截图配置项 | 截图当前值 | 远程指令 | 设置语法 | 查询语法 |
| --- | --- | --- | --- | --- |
| 内部参考信号频率(Hz) | Ch-A = 1.000 / Ch-B = 102000.000(灰显) | `FREQD` | `FREQD i,f <lf>` | `FREQD? i <lf>` |

### 参数说明

- `i` = 通道
- `f` = 频率值(单位 Hz)
- 范围:0 ~ 102 kHz(说明书 4.2.4 注明实际可用 1 mHz~102 kHz)
- 最小分辨率:1 mHz
- 默认值:1.000 kHz

> 📝 指令说明:`FREQD? 1` 会返回通道 A **当前的实际参考信号频率**(内部或外部锁相后)。

**配置示例**:
```text
FREQD 1,10E3 <lf>     ; Ch-A 内部频率 = 10 kHz
FREQD? 1 <lf>         ; 返回 "10000.000" 或当前实际频率
```

## 附录:本节截图 → 完整指令还原(可直接复制执行)

```text
; ===== Ch-A(内部参考, 1 kHz)=====
FMODD 1,1 <lf>        ; 参考源 = 内部参考
FREQD 1,1.000 <lf>    ; 内部频率 = 1.000 Hz (截图显示,实际默认 1.000 kHz)
PHASD 1,0.00 <lf>     ; 相位 = 0.00°
; 以下为 Ch-A 灰显项(因未选 Internal Sweep),指令无限制可正常下发:
SWTPD 1,0 <lf>        ; 扫频类型 = Linear
SLLMD 1,1000.000 <lf> ; 扫频开始 = 1000.000 Hz
SULMD 1,1000.000 <lf> ; 扫频截止 = 1000.000 Hz
SSLLD 1,1000.000 <lf> ; 步进 = 1000.000 Hz
STLMD 1,1 <lf>        ; 步进时间 = 1 ms
SWRMD 1,0 <lf>        ; 扫频运行 = 停止

; ===== Ch-B(外部参考, 手册原文: 过零检测)=====
FMODD 2,0 <lf>        ; 参考源 = 外部参考
RSLPD 2,1 <lf>        ; 手册原文: 外部类型 = 正弦波过零检测
PHASD 2,0.00 <lf>     ; 相位 = 0.00°
; 以下为 Ch-B 灰显项(因参考源 = External),指令无限制:
SWTPD 2,0 <lf>
SLLMD 2,1000.000 <lf>
SULMD 2,0.001 <lf>
SSLLD 2,0.001 <lf>
STLMD 2,1 <lf>
SWRMD 2,0 <lf>
```

> 注：本节中的 `RSLPD 2,1` 仅表示手册原文写法，不代表当前真机锁 PLL 时的最终观测状态。当前真机锁定态请以 `RSLPD? 2 = 2` 的只读产物为准。
