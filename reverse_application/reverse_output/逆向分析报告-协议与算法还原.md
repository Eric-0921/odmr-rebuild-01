## 二、串口通信协议（SCPI 指令集）

### 2.1 串口参数

| 参数 | 值 |
|---|---|
| 波特率 | 9600（可配置：1200~115200） |
| 数据位 | 8 |
| 停止位 | 1 |
| 校验 | None |
| 流控 | None |
| DTR | true |
| 读超时 | 100 ms |

### 2.2 连接初始化流程

```
1. 打开串口
2. 发送: *IDN?\n          → 验证设备身份（期望返回 MAYNUO,... 格式）
3. 发送: SYST:REM\n       → 切换远程控制模式
4. 发送: VOLT 75\n        → 设定输出电压 75V
5. 发送: CURR 0\n         → 设定输出电流 0A
6. 发送: OUTP 0\n         → 关闭输出
7. 注册 DataReceived 事件，启动 timer1（300ms）
```

### 2.3 断开连接流程

```
1. 发送: SYST:LOC\n       → 切换本地模式
2. 关闭串口
```

### 2.4 实时控制指令

| 指令 | 说明 | 触发条件 |
|---|---|---|
| `CURR {value}\n` | 设定输出电流（单位 **A**，保留5位小数） | 零偏/复现电流值改变时 |
| `OUTP 1\n` | 开启电源输出 | 用户点击"输出"开关 |
| `OUTP 0\n` | 关闭电源输出 | 用户关闭"输出"开关 |
| `MEAS:CURR?\n` | 查询实际输出电流 | timer1 定时发送（300ms/500ms） |

> **重要单位转换**：软件内部电流单位为 **mA**，发送给电源时转换为 **A**（`÷1000`）。电源回传值单位为 **A**，软件接收后转换为 **mA**（`×1000`）。

### 2.5 电流设置公式

```csharp
string cmd = $"CURR {Math.Abs(totalCurr) / 1000.0:f5}\n";
```

- `totalCurr`：总输出电流，单位 **mA**
- `Math.Abs()`：只发送正值（Maynuo M8812 只支持正电流设定，磁场方向通过线圈绕向固定）
- 格式：`f5` = 保留5位小数

## 三、磁场控制算法还原

### 3.1 核心公式

| 换算方向 | 公式 | 单位 |
|---|---|---|
| 电流 → 磁场 | `Mag(nT) = Curr(mA) × CoilConstant(nT/mA)` | 输入 mA，输出 nT |
| 磁场 → 电流 | `Curr(mA) = Mag(nT) / CoilConstant(nT/mA)` | 输入 nT，输出 mA |

### 3.2 工作模式

软件有两种互相关联的工作状态：

#### 模式 A：不锁定零场（LockZero = OFF）

| 控件 | 作用 |
|---|---|
| `zeroSetCurr` | 直接设定并输出零场偏置电流 |
| `recurSetCurr` / `recurSetMag` | 仅做计算显示，**不影响输出** |

- 输出电流 = `zeroSetCurr`
- 回读总电流显示在 `textBoxTotalCurr` 和 `textBoxZeroCurr`

#### 模式 B：锁定零场（LockZero = ON）+ 输出开启

- 输出电流 = `zeroSetCurr + recurSetCurr`（零偏 + 复现）
- `recurSetCurr` 改变 → 自动重新计算并发送 `CURR` 指令
- `recurSetMag` 改变 → 先换算为电流 `curr = mag / coilConstant`，再输出

### 3.3 安全保护逻辑

```
POWER_MAX_CURR = 5000.0 mA（5A）

如果 zeroSetCurr + recurSetCurr > 5000 mA:
    → 弹出警告"总电流超范围！"
    → 自动限制 recurSetCurr = 5000 - zeroSetCurr
    → 发送 CURR 5.00000
```

### 3.4 显示值计算（DataReceived 事件）

当收到 `MEAS:CURR?` 的响应时（单位 A）：

```csharp
double totalCurr_mA = result * 1000.0;          // 总电流回读值（mA）
textBoxTotalCurr.Text = totalCurr_mA.ToString("f2");

if (LockZero == ON && Output == ON) {
    double recurCurr = totalCurr_mA - zeroSetCurr;   // 复现电流 = 总电流 - 零偏
    double recurMag  = recurCurr * coilConstant;      // 复现磁场
    textBoxRecurCurr.Text = recurCurr.ToString("f2");
    textBoxRecurMag.Text  = recurMag.ToString("f2");
    textBoxZeroCurr.Text  = zeroSetCurr.ToString("f2");
} else {
    textBoxZeroCurr.Text = totalCurr_mA.ToString("f2");  // 零偏回读 = 总电流
}
```

## 四、数据保存机制

### 4.1 文件格式

| 项目 | 说明 |
|---|---|
| 文件名 | `yyyyMMddHHmmss.txt` |
| 编码 | 默认（UTF-8 BOM 或 ANSI，取决于 StreamWriter） |
| 扩展名 | `.txt`（实际为 CSV 格式） |

### 4.2 CSV 头部

```
时间,x零场偏置,x复现电流,x复现磁场磁场值,y零场偏置,y复现电流,y复现磁场磁场值,z零场偏置,z复现电流,z复现磁场磁场值
```

### 4.3 数据行格式

```
yyyyMMddHHmmss,zeroX,recurCurrX,recurMagX,zeroY,recurCurrY,recurMagY,zeroZ,recurCurrZ,recurMagZ
```

示例：
```
20260530180000,100.00,50.00,7150.00,200.00,30.00,4253.10,150.00,40.00,6246.00
```

### 4.4 采样间隔

- 由 `numericUpDown1` 控制，范围 **0.2s ~ 60s**，默认 **1s**
- 实际保存由 `timer2`（500ms）驱动，在指示灯熄灭时写入一行
- 注意：`timer1` 的轮询间隔在保存模式下会被同步改为采样间隔值

## 八、逆向结论

1. **无混淆/无保护**：两个程序集均为标准 .NET 程序，类型名、方法名、变量名全部清晰可读。
2. **协议简单标准**：使用 Maynuo 电源的标准 SCPI 子集（`SYST`, `VOLT`, `CURR`, `OUTP`, `MEAS`），无任何自定义协议封装，已**经实际硬件验证**。

4. **已具备独立控制条件**：基于本报告，可以直接用 Rust 编写替代控制程序，无需依赖原有 UI。

## 九、附：Rust 独立控制实现草案

以下为基于逆向结果和验证数据编写的 Rust 协议层代码，可直接作为替代控制程序的基础。

```rust
use serialport::{SerialPort, ClearBuffer};
use std::time::Duration;
use std::io::{Read, Write};

/// 单轴电源配置
#[derive(Debug, Clone)]
pub struct AxisConfig {
    pub name: &'static str,       // "X", "Y", "Z"
    pub port: String,             // e.g. "COM4"
    pub coil_constant: f64,       // nT/mA
    pub sn_tail: &'static str,    // 用于校验，如 "2020"
}

/// Maynuo M8812 电源控制器
pub struct PowerController {
    port: Box<dyn SerialPort>,
    cfg: AxisConfig,
}

/// 安全限制
const POWER_MAX_CURR_MA: f64 = 5000.0;
const VOLTAGE: f64 = 75.0;

impl PowerController {
    /// 打开串口并验证设备身份
    pub fn open(cfg: AxisConfig) -> Result<Self, Box<dyn std::error::Error>> {
        let port = serialport::new(&cfg.port, 9600)
            .data_bits(serialport::DataBits::Eight)
            .parity(serialport::Parity::None)
            .stop_bits(serialport::StopBits::One)
            .timeout(Duration::from_millis(300))
            .open()?;

        let mut ctrl = Self { port, cfg };
        ctrl.identify()?;
        Ok(ctrl)
    }

    fn write_cmd(&mut self, cmd: &str) -> Result<(), Box<dyn std::error::Error>> {
        let line = if cmd.ends_with('\n') {
            cmd.to_string()
        } else {
            format!("{}\n", cmd)
        };
        self.port.write_all(line.as_bytes())?;
        self.port.flush()?;
        Ok(())
    }

    fn read_response(&mut self) -> Result<String, Box<dyn std::error::Error>> {
        let mut buf = vec![0u8; 256];
        let n = self.port.read(&mut buf)?;
        Ok(String::from_utf8_lossy(&buf[..n]).trim().to_string())
    }

    /// 发送 *IDN? 并验证返回
    pub fn identify(&mut self) -> Result<String, Box<dyn std::error::Error>> {
        self.port.clear(ClearBuffer::Input)?;
        self.write_cmd("*IDN?")?;
        std::thread::sleep(Duration::from_millis(200));
        let resp = self.read_response()?;
        if !resp.to_uppercase().contains("MAYNUO") {
            return Err(format!("{}轴设备身份异常: {}", self.cfg.name, resp).into());
        }
        Ok(resp)
    }

    /// 连接初始化（SYST:REM → VOLT 75 → CURR 0 → OUTP 0）
    pub fn init(&mut self) -> Result<(), Box<dyn std::error::Error>> {
        self.write_cmd("SYST:REM")?;
        std::thread::sleep(Duration::from_millis(50));
        self.write_cmd(&format!("VOLT {}", VOLTAGE))?;
        std::thread::sleep(Duration::from_millis(50));
        self.write_cmd("CURR 0")?;
        std::thread::sleep(Duration::from_millis(50));
        self.write_cmd("OUTP 0")?;
        std::thread::sleep(Duration::from_millis(50));
        Ok(())
    }

    /// 安全断开（CURR 0 → OUTP 0 → SYST:LOC）
    pub fn shutdown(&mut self) -> Result<(), Box<dyn std::error::Error>> {
        let _ = self.set_current_ma(0.0);
        let _ = self.set_output(false);
        let _ = self.write_cmd("SYST:LOC");
        std::thread::sleep(Duration::from_millis(50));
        Ok(())
    }

    /// 设置输出电流（单位 mA），内部转换为 A
    pub fn set_current_ma(&mut self, curr_ma: f64) -> Result<(), Box<dyn std::error::Error>> {
        if curr_ma < 0.0 || curr_ma > POWER_MAX_CURR_MA {
            return Err(format!("电流 {} mA 超出安全范围 [0, {}]", curr_ma, POWER_MAX_CURR_MA).into());
        }
        let curr_a = curr_ma.abs() / 1000.0;
        self.write_cmd(&format!("CURR {:.5}", curr_a))?;
        Ok(())
    }

    /// 开关输出
    pub fn set_output(&mut self, on: bool) -> Result<(), Box<dyn std::error::Error>> {
        self.write_cmd(&format!("OUTP {}", if on { 1 } else { 0 }))?;
        Ok(())
    }

    /// 查询实际输出电流，返回 mA
    pub fn read_current_ma(&mut self) -> Result<f64, Box<dyn std::error::Error>> {
        self.port.clear(ClearBuffer::Input)?;
        self.write_cmd("MEAS:CURR?")?;
        std::thread::sleep(Duration::from_millis(200));
        let resp = self.read_response()?;
        let curr_a: f64 = resp.parse()
            .map_err(|e| format!("解析电流失败 '{}': {}", resp, e))?;
        Ok(curr_a * 1000.0)
    }

    /// 磁场(nT) → 电流(mA)
    pub fn mag_to_curr(&self, mag_nt: f64) -> f64 {
        mag_nt / self.cfg.coil_constant
    }

    /// 电流(mA) → 磁场(nT)
    pub fn curr_to_mag(&self, curr_ma: f64) -> f64 {
        curr_ma * self.cfg.coil_constant
    }

    /// 输出总电流 = 零偏 + 复现（锁定零场模式）
    pub fn calc_total_current(
        zero_offset_ma: f64,
        recur_curr_ma: f64,
    ) -> Result<f64, String> {
        let total = zero_offset_ma + recur_curr_ma;
        if total > POWER_MAX_CURR_MA {
            return Err(format!("总电流 {} mA 超过上限 {} mA", total, POWER_MAX_CURR_MA));
        }
        Ok(total)
    }
}

impl Drop for PowerController {
    fn drop(&mut self) {
        let _ = self.shutdown();
    }
}

// ========== 使用示例 ==========
fn main() -> Result<(), Box<dyn std::error::Error>> {
    let cfg = AxisConfig {
        name: "X",
        port: "COM4".to_string(),
        coil_constant: 143.26,
        sn_tail: "2020",
    };

    let mut ctrl = PowerController::open(cfg)?;
    ctrl.init()?;

    // 设置零偏 100mA，开启输出
    ctrl.set_current_ma(100.0)?;
    ctrl.set_output(true)?;
    std::thread::sleep(Duration::from_millis(200));

    // 回读实际电流
    let actual = ctrl.read_current_ma()?;
    println!("实际输出: {:.2} mA", actual);

    // 锁定零场后追加复现磁场电流
    let recur_mag_nt = 1000.0; // 目标复现磁场 1000 nT
    let recur_curr = ctrl.mag_to_curr(recur_mag_nt);
    let total = PowerController::calc_total_current(100.0, recur_curr)?;
    ctrl.set_current_ma(total)?;
    println!("设定总电流: {:.2} mA (零偏100 + 复现{:.2})", total, recur_curr);

    // 程序结束自动 shutdown（Drop）
    Ok(())
}
```
