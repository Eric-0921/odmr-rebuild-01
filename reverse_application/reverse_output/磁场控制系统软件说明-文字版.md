使用说明 
一、软件安装 
本软件的发行方式为绿色免安装版，拷贝到计算机后，直接运行
即可。 
支持的操作系统： 
Windows7 SP1 及以上、windows8、windows10、windows11。 
运行软件前，需安装的先决条件（安装包已随软件一同交付）： 
1. 安装VC++运行时 
大多情况下，计算机已安装有VC++运行时，无需再次安装，
可直接安装.NET Framework 4.8。当提示缺少VC++运行时，再
运行安装程序VC_redist.x64进行安装。 
2. 安装.NET Framework 4.8 运行时 
运行ndp48-x86-x64-allos-enu.exe 按步骤完成安装。 
二、连接设备操作步骤 
连接电源前，应先进行“连接设置”和“参数设置”，并确保电源已可
靠连接，然后点击“连接设备”。操作步骤见下图： 
图 1连接设备步骤
